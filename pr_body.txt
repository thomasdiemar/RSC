## Setup Instructions

### For Local Development
1. Create/update .env with your PAT:
   ```
   AZURE_DEVOPS_NUGET_TOKEN=<your-pat-here>
   ```
2. Run scripts:
   ```powershell
   .\build.ps1  # Full workflow
   .\test.ps1   # Tests only
   ```

### For CI/CD (GitHub Actions)
1. Configure secret in GitHub: Settings > Secrets > AZURE_DEVOPS_NUGET_TOKEN
2. Workflows use the secret automatically
3. No PAT needed in config files

## Azure DevOps PAT Requirements
- Scope: Packaging (Read & Write)
- Permissions: Read/publish packages, access vulnerability data
- Creation: Azure DevOps > User Settings > Personal Access Tokens
