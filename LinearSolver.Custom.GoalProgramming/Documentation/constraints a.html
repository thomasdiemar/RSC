Handling the equality t₁ = t₂ in a Preemptive Bounded Simplex Tableau

We keep bounds inside the simplex and represent the equality as a real tableau row. Below, three small scenarios show: (A) a row pivot (θ>0), (B) a preemptive bound hit (no pivot), and (C) a degenerate row pivot (θ=0). All three enforce t₁ = t₂ without substitutions and with full LB/UB logic.

Common setup (applies to all scenarios)
  • Equality row kept inside the basis: t₁ − t₂ = 0.
  • Objective (minimization): z = −(t₁ + t₂) (fuel proxy).
  • Reduced-cost rule (min): LB nonbasic can enter if r < 0, UB nonbasic if r > 0.
  • At the start we take the equality row as basic (solving for t₁), nonbasic t₂ at LB.
  • Because of the coupling, pricing typically gives r(t₂) = -2 (one unit from t₂ itself + one unit from induced basic t₁ move).
Scenario A — Row pivot (θ > 0): both have room; we pivot on the equality row
Bounds: 0 ≤ t₁ ≤ 1, 0 ≤ t₂ ≤ 1. Start at t₁=t₂=0 (feasible).

ART (step limits)
Direction in basic space: the t₂-column in this row is −1 ⇒ d = −1.
Basics move: Δt₁ = −d·θ = +θ, entering moves: Δt₂ = +θ.
Row limit: t₁ can increase up to UB 1 ⇒ θ_row = (1−0)/1 = 1.
Opposite bound of t₂: θ_bound = (1−0)=1.
Tie ⇒ row pivot at θ=1.
Normalize pivot row: pivot element is −1 ⇒ divide the row by −1 to make it +1 in column t₂.

Scenario C — Degenerate row pivot (θ = 0): equality row blocks immediately
Bounds: t₁ is fixed at 0 (LB=UB=0), 0 ≤ t₂ ≤ 1. Start at t=0.

ART: d = −1 ⇒ Δt₁=+θ. But t₁ is fixed at 0: first hit is UB=0 with θ_row = 0. Opposite bound of t₂: θ_bound = 1. Row wins at θ=0 ⇒ degenerate row pivot.
Normalize & pivot (θ=0): divide the pivot row by −1 to make +1 in col t₂; zero that column elsewhere; relabel row’s basic from t₁→t₂.

The basis changed (can unblock pricing later), but the point stayed at t=0 because θ=0. The equality t₁=t₂ is still enforced; the row now solves for t₂ instead of t₁.

What to remember
  • Equality as a row: Keep t₁ − t₂ = 0 as a tableau row with a basic variable (no substitution).
  • Coupled movement: When t₂ enters, the equality row forces the basic (e.g., t₁) to move with it (the direction column).
  • ART outcomes: (i) row pivot, (ii) bound hit (no pivot), (iii) degenerate pivot (θ=0). Equality is preserved in all three.
  • Preemptive stages: Once a priority cancels torque via these equalities, keep those rows in the basis for lower priorities (“locked” constraints).
